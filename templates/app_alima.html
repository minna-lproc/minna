Alima is a mobile application that has features that can help preserve any language in the Philippines. It has three user types. The basic user has the ability to use most of the functions of the application and they also contribute text and image data for the community page and they can also contribute text and audio data that will be sent to the validators so that it can be verified and be used in the dictionary feature of the application. Basic users can also apply as a validator so that they can help the application grow even further. The validator user type is just like the basic user however they have the ability to accept or decline the contributions of the user.
